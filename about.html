Using the Tutorial Template

This site is a clean starter for publishing tutorials. Content lives in Markdown, the layout is prewired for math and code, and you only need to point the template page at your files.

Add a tutorial

  1. Create a folder under content/tutorials/ (for example, content/tutorials/getting-started/).
  2. Add a Markdown file inside that folder (for example, lesson.md). Images can live in the same folder.
  3. Open tutorial.html?title=Getting%20Started&md=content/tutorials/getting-started/lesson.md. The query string tells the page which Markdown file to load.
  4. Use H1, H2, and H3 headings in your Markdown - the sidebar is generated from them automatically.

Homepage tips

  • Duplicate a card block in index.html for every tutorial you want to feature.
  • Change the card text and link to a tutorial.html?md=... URL that matches your Markdown file.
  • Use the CSS variables in styles.css to adjust colors and spacing quickly.

Markdown helpers

Use these patterns to trigger the built-in styles:

  • Callouts: start a blockquote with > **Question:**, > **Tip:**, or > **Warning:**.
  • Math: wrap inline math with $...$ and block math with $$...$$ (MathJax is already configured).
  • Code: fenced code blocks are highlighted automatically by Highlight.js.

Where to customize

  • styles.css: landing page layout and palette.
  • css/tutorial.css: tutorial page layout, sidebar, and typography.
  • js/markdown-callback-loader.js: Markdown rendering behavior.
